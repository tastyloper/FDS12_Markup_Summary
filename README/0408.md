## HTML

### dl, dt, dd

* dl은‘Definition List’의 약자로 용어를 설명하는 목록
  * html 5.1부터 dl안에 모든 것을 감싸는 div를 넣을 수 있다. 단, dt와 dd를 포함해야한다.
* dt는 'Definition term'의 약자로 용어의 제목
* dd는 'Definition description'의 약자로 용어를 설명
